Process

Step by step of this GenArt Program creation

Starting from scratch the first step was creating many objects (circles, squares, and text) inside for loops to visualize how random objects work together.
step 00 have many objects inside for loops

Challenge 1
With a lot of iterations, and using the functions push(), translation(), and pop() I was able to solve the random rotation around the object's own axis.

Challenge 2
I wanted to use random color palettes, so I decided to use a Jason database from this project with a beautiful color set to use. Here is where I realized I needed to combine For-loops with If-else statements:
step 01 random pallete step 02 random pallete

Then, I was able to include radio buttons and edit values from that selection so the circle sizes could be adjusted to create different variations. At this stage, I struggled with resetting the canvas.
step 03 shapes fixed but not resetting

Challenge 3
After many hits and trying different things, I discovered that I needed to reset the Arrays within the NewArt function so I was able to create a new artwork every time the button New Art is pressed. In this version, I am also including several images for the selection.
step 04 adding images
